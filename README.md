# Video Reels Preview

Редактор обложек для видео. Захват кадра из YouTube/Rutube/MP4 → редактирование на canvas (текст, фигуры, фильтры, слои) → экспорт в PNG/JPEG/WebP. Всё работает локально в браузере, файлы не отправляются на сервер.

**[🚀 Открыть →](https://tom-opencart.github.io/video-reels-preview/)**

## Возможности

- **YouTube, Rutube, MP4/WebM** — вставьте ссылку или загрузите файл
- **Захват кадра** — выберите момент на таймлайне и нажмите «Снять кадр»
- **Инструменты** — Текст, Прямоугольник, Круг, Стрелка, Линия
- **Слои** — управление порядком, видимостью, блокировкой, drag-and-drop
- **Фильтры** — яркость, контраст, насыщенность, размытие, сепия, ч/б, инверсия
- **Undo/Redo** — Ctrl+Z / Ctrl+Shift+Z, до 50 шагов
- **Пресеты размеров** — YouTube Shorts (9:16), Instagram (1:1), YouTube (16:9), кастомный
- **Шаблоны** — сохранение/загрузка через localStorage, экспорт/импорт JSON
- **Экспорт** — PNG, JPEG, WebP, копирование в буфер, предпросмотр

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `V` | Выделение |
| `T` | Текст |
| `R` | Прямоугольник |
| `C` | Круг |
| `A` | Стрелка |
| `L` | Линия |
| `Ctrl+Z` | Отменить |
| `Ctrl+Shift+Z` | Повторить |
| `Ctrl+E` | Экспорт |
| `Ctrl+C` | Копировать в буфер |
| `Ctrl+колесо` | Зум |
| `Delete` | Удалить слой |
| `Escape` | Снять выделение |
| `Double-click по тексту` | Редактировать текст |

## Структура

```
video-reels-preview/
├── index.html          ← Точка входа
├── css/
│   └── style.css       ← Стили, CSS-переменные, адаптив

│   ├── filters.js      ← Фильтры изображения
│   ├── history.js      ← Undo/Redo
│   ├── presets.js      ← Пресеты размеров
│   ├── templates.js    ← Шаблоны (localStorage)
│   ├── export.js       ← Экспорт и предпросмотр

```

## Технологии

Чистый HTML/CSS/JS (ES modules), без зависимостей и сборки. Открывается в любом браузере.
