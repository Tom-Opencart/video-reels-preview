# Video Reels Preview

Мини-редактор обложек для видео. Захват кадра из YouTube/Rutube/MP4 → редактирование (текст, фигуры, фильтры, слои) → экспорт.

**[Открыть →](https://tom-opencart.github.io/video-reels-preview/)**

## Возможности

- **YouTube, Rutube, MP4/WebM** — вставьте ссылку или загрузите файл
- **Захват кадра** — выберите момент и нажмите «Снять кадр»
- **Canvas-редактор** — полноценный редактор обложек
- **Текст** — шрифт, размер, цвет, фон, тень, жирный/курсив
- **Фигуры** — прямоугольник, круг, стрелка, линия
- **Слои** — порядок, видимость, блокировка, drag-and-drop
- **Фильтры** — яркость, контраст, насыщенность, сепия, ч/б
- **Undo/Redo** — Ctrl+Z / Ctrl+Shift+Z
- **Пресеты** — YouTube Shorts (9:16), Instagram (1:1), YouTube (16:9), кастомный
- **Шаблоны** — сохранение/загрузка через localStorage
- **Экспорт** — PNG, JPEG, WebP + копирование в буфер
- **Мобильная версия** — адаптивный layout с нижней панелью

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `V` | Выделение |
| `T` | Текст |
| `R` | Прямоугольник |
| `C` | Круг |
| `A` | Стрелка |
| `L` | Линия |
| `Ctrl+Z` | Отменить |
| `Ctrl+Shift+Z` | Повторить |
| `Delete` | Удалить слой |
| `Ctrl+D` | Дублировать |
| `Ctrl+E` | Экспорт |
| `Ctrl+колёсико` | Зум |
| `Escape` | Снять выделение |

## Технологии

Чистый HTML/CSS/JS, ES modules, без зависимостей. Работает в любом современном браузере.
