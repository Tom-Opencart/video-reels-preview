# Video Reels Preview

Генератор превью-изображений из видео. Всё работает локально в браузере, файлы не отправляются на сервер.

**[🚀 Открыть →](https://tom-opencart.github.io/video-reels-preview/)**

## Возможности

- **YouTube, Rutube, MP4/WebM** — вставьте ссылку или загрузите файл
- **Захват кадра** — выберите момент на таймлайне и нажмите «Снять кадр»
- **Текст** — добавляйте надписи с настраиваемым шрифтом, цветом, фоном и прозрачностью
- **Зум** — Ctrl+колесо для приближения, зажать и перетащить для перемещения
- **Предпросмотр** — полноэкранный показ результата перед скачиванием
- **Скачать / Копировать** — экспорт в PNG, JPEG, WebP или в буфер обмена

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl + колесо` | Зум превью |
| `Зажать + тащить` | Перемещение при зуме |
| `Enter` | Сохранить текст |
| `Escape` | Выйти из редактирования / закрыть предпросмотр |
| `Delete` | Удалить выделенный текст |
| `Пробел` | Play / Pause видео |

## Технологии

Чистый HTML/CSS/JS, без зависимостей. Открывается в любом браузере.
